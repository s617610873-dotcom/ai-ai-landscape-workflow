一、整体架构

三个对话 + WPS 合稿，形成完整的方案生产管线：

┌─ 对话1: PDF 分析 ──────────────────────────────────────────┐
│ 上传参考方案PDF→01模板→框架描述.txt / 上传项目PDF→02模板→项目数据.txt │
└──────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─ 对话2: 小流程出图 ─────────────────────────────────────────┐
│ 03模板+项目数据→5类图纸AB双版→逐张出图→S01_B~S05_B+S01_A~S05_A │
└──────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─ 对话3: 大流程内容+补图 ────────────────────────────────────┐
│ 04模板→分页内容 → 05模板→补图提示词→P01_A~P0N_B              │
└──────────────────────────────────────────────────────────────┘
                          │
                          ▼
                    WPS 合稿: B版纯图+A版排版参考

二、配方库（5 个模板文件）

文件1：01_提取参考方案框架.txt

用途：从参考方案 PDF 提取完整文档结构和文本风格。
使用场景：对话1，上传 PDF 后直接发。

请分析我上传的这套完整的景观设计方案PDF（含所有页面），
按以下权重顺序提取参考框架，输出纯文本，不要markdown标记。

第一部分：文档结构（权重最高，必须完整）
- 总共有多少页
- 逐页列出：页码 + 页面类型
  （封面/目录/设计概念/总平面图/分析图/节点效果图/剖面图/植物配置/小品设施/区位分析/人群分析/专项设计/封底/其他）

文件2：02_提取项目数据.txt

用途：从项目 PDF 提取结构化数据。
使用场景：对话1，接着上一条消息发。

请从以下项目PDF的文字内容中提取景观设计所需的关键信息，
按下面10个字段输出纯文本（每行一个字段，不要任何格式标记、不要列表符号）：
1. 项目名称：2. 红线面积（ha）：3. 建筑布局简述：...

文件3：03_小流程出图指令.txt

用途：生成小流程5类核心图纸的 AB 双版提示词。
使用场景：对话2，粘贴项目数据后发。

请基于以下【项目数据】和【参考框架（可选）】，生成一套景观设计核心图纸
的 GPT Image 2 提示词（英文）。每张图出两个版本：
- Version A：带标注/图例/文字说明的交付版
- Version B：纯图版，无文字，用作WPS素材

文件4：04_生成大流程分页内容.txt

用途：生成完整汇报PPT的每页内容，并与小流程图纸做映射。
使用场景：对话3，粘贴框架+数据+映射规则后发。

=== 第一步：图纸映射（必做） ===
S01=现状平面图 S02=彩平图 S03=总平面图 S04=分析图 S05=效果图/鸟瞰图
逐页判断：PAGE N → 复用 S0X_B / 需新出图

文件5：05_大流程补图提示词.txt

用途：仅对标注"需新出图"的页面生成 AB 双版提示词。
使用场景：对话3，紧接着上一条消息发。

为每一页"需新出图"页面生成一对 GPT Image 2 提示词（英文）
Version A：完整方案页含中文文字
Version B：纯图版，无文字

三、三组对话的规划

对话1：PDF 分析

操作开新对话？为什么
上传参考方案PDF + 发01模板新对话 1上传文件需要干净上下文
上传项目PDF + 发02模板不另开同一个分析会话
下载保存——

对话2：小流程出图

操作开新对话？为什么
粘贴项目数据 + 发03模板新对话 2从分析切换到图像生成
逐张出图及修改不另开修改依赖上下文

对话3：大流程内容 + 补图

操作开新对话？为什么
粘贴框架 + 数据 + 04模板新对话 3切换到方案内容规划
继续发05模板不另开05依赖04的映射结果

WPS 合稿（离线）

操作
新建PPT → 铺B版图 → 叠原生文本框 → 套母版 → 微调

四、一句话理清对话策略

对话1 只读 → 读参考方案、读项目数据 → 输出文本

开新对话

对话2 只画 → 画小流程5张核心图 → 输出 AB 双版 PNG

开新对话

对话3 先规划再补画 → 先做映射+分页内容 → 同一个对话里补画缺的页面

对话3 不能分裂——05依赖04的映射结果，开新对话AI就忘了
